# 简介 #

> Windows下列出一个ftp服务器上所有文件列表的小程序. 支持IPv6的FTP列表程序

# 特点 #
  1. 同时支持IPv4和支持IPv6

# 使用说明 #

ftplist.exe **_host`[`:port`]`_**  **_username_** **_password_**

# 系统要求 #
> 由于本程序是用C#编写的，所以需要 .Net Framework
